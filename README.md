## 🚀 How to Use

### Option 1 — Open Locally
```bash
# Just open the file in any browser
open thorman-creatives.html
```

### Option 2 — Deploy on Netlify (Free)
1. Go to [netlify.com](https://netlify.com)
2. Drag and drop `thorman-creatives.html`
3. Your site is live in seconds ✅

### Option 3 — GitHub Pages
```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/thormancreatives.git
git push -u origin main
```
Then go to **Settings → Pages → Deploy from main branch** ✅

## 📧 Contact Form Setup (Optional)

To make the contact form send real emails, use **Formspree** (free):

1. Sign up at [formspree.io](https://formspree.io)
2. Create a new form → copy your Form ID (e.g. `xpwzjkrq`)
3. In `thorman-creatives.html`, find:
   ```html
   action="https://formspree.io/f/YOUR_FORM_ID"
   ```
4. Replace `YOUR_FORM_ID` with your actual ID
5. All submissions will go directly to `thormancreatives@gmail.com` ✅
